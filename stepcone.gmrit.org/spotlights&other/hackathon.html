Hackathon

Organized By

Department of Computer
Science&Engineering

Hackathon is a coding event in which computer programmers, problem solvers, coding geeks and others involved in software development, collaborates intensively on problem solution with their coding skills. A vast majority of problems appearing in programming contests are mathematical or logical in nature. Irrespective of the problem category, the process of solving a problem can be divided into two broad steps, constructing an efficient algorithm, and implementing the algorithm in a suitable programming language. Contestants are referred to as sport programmers. Competitive programming is recognized and supported by several multinational software and IT companies. The aim of competitive programming is to write source code of computer programs which are able to solve given problems.
